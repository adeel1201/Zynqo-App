# **App Name**: Zynqo

## Core Features:

- Secure Multi-Protocol Identity: Unified authentication system allowing users to register and sign in via phone OTP, email/password, or social providers like Google and Facebook, managed securely with unique identifiers.
- Omni-Channel Messaging Hub: A high-concurrency real-time messaging interface supporting private chats and group threads with features like read receipts, typing indicators, and file sharing using a backend storage and message queue.
- AI Smart-Reply & Translation Tool: A generative AI feature that analyzes message context to suggest contextual replies and offers real-time language translation for global communication within chat threads.
- Epicenter Moments Feed: A dynamic social discovery engine where users post rich-media 'Moments' with liking, commenting, and configurable privacy visibility rules (Public, Friends, Private).
- Ephemeral Status Stories: 24-hour expiring status updates for text, photos, and videos with viewer list tracking and audience privacy controls.
- Proximity-Based Discovery: Leverages real-time geolocation to find nearby users, displaying relative distances and facilitating new connections based on local interaction parameters.
- Firestore Dynamic Synchronization: Real-time state synchronization for presence indicators, unread message counters, and instant notifications using Firestore's scalable cloud architecture.

## Style Guidelines:

- The design follows a dark, 'Midnight Premium' aesthetic to foster deep immersion. The primary color is a vibrant 'Electric Amethyst' (#9F5FF5), providing a high-tech energy. The background is a deeply desaturated 'Onyx Void' (#0E0C12), providing maximum contrast for vibrant media.
- Accent: An analogous 'Digital Cobalt' (#5F87F5) is used for active states, CTA borders, and indicators to maintain a harmonious cool-tone palette.
- Font pairing for a technical, modern feel: 'Space Grotesk' is used for high-impact headlines and UI controls (sans-serif), while 'Inter' is utilized for long-form messaging and feed text to ensure maximum readability across screen sizes.
- Feather-light, line-style icons with neon accents when active, ensuring the interface remains uncluttered and futuristic.
- A sleek 6-tab bottom navigation provides immediate access to Core Chats, Status, Calls, Moments, Discover, and Profile. Messaging views use high density 'bubble-less' spacing inspired by modern minimalist design.
- Liquid transitions between tabs and 'spring-physics' for feed scrolling and photo expanding to create a high-performance native-feel experience.
